# Healthcare-Analytics
Healthcare Analytics is an end-to-end data analysis project focused on improving clinical decision-making, patient outcomes, and operational efficiency using data-driven insights.
The project analyzes hospital data such as patient demographics, medical conditions, treatments, hospital stay duration, and resource utilization to identify meaningful trends and performance indicators.

### Dataset
<a href="https://github.com/elite9510/healthcare-Analytics/blob/main/Papollo-Healtcare-Dataset.xlsx">Datasets</a>
### process
Analyze patient data to understand admission trends
Identify factors affecting patient outcomes and hospital stay length
Track healthcare KPIs such as readmission rate, bed occupancy, and treatment success rate
Visualize hospital operations and patient flow
Provide recommendations to improve service quality and hospital efficiency
